Basic information:

This application sorts the following shapes regarding their Heights, Base Area and Volume: 

1. Cylinder
2. Cone
3. Prisms
  - Square Base
  - Equilateral Triangle Base
  - Pentagon Base
  - Octagon Base
4. Pyramids

The following algorithms are used: 

1. Bubble sort
2. Selection sort
3. Insertion sort
4. Merge sort
5. Quick sort
6. Our own choice of sorting algorithm
